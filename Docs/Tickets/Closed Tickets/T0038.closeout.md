# Ticket Closeout: T0038 - Open incident workspace v1

## What shipped
- Added a first-class Open Incident Workspace for creating and managing unsolved incidents.
- Added incident records with offense window, incident metadata, and one or more scene locations with radius.
- Added incident cross-reference v1 against messages and locations using offense-window and radius filtering.
- Added timeline integration so incidents can act as timeline anchors/markers.
- Added the ability to pin/save relevant findings to the incident workspace.
- Preserved provenance behavior for cross-referenced results.
- Fixed a SQLite-specific refresh bug by making incident ordering SQLite-safe when loading incident lists.
