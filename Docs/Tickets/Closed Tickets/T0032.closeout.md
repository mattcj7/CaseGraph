# Ticket Closeout: T0032 - Performance instrumentation and budgets

## What shipped
- Added a reusable performance instrumentation layer with shared budget thresholds and structured timing events.
- Standardized performance logging across startup, case-open, feature readiness/open, query execution, and practical maintenance flows.
- Added uniform `PerformanceOperationStarted`, `PerformanceOperationCompleted`, and `PerformanceSlowPathWarning` events.
- Propagated correlation id, case context, and operation labeling through the instrumentation layer.
- Wired instrumentation into startup, case-open, Search/Timeline/Reports/Incident Window flows, and relevant maintenance/query services.
- Added deterministic regression coverage for instrumentation behavior and budget-based slow-path warnings.
