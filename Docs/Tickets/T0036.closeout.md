# Ticket Closeout: T0036 - Synthetic evidence generator window

## What shipped
- Added a standalone `CaseGraph.SyntheticDataGenerator` WPF utility with a simple generator window for seed, dataset count, person count, approximate message count, approximate location count, required `offense_window` profile selection, and output-folder selection.
- Added a deterministic synthetic dataset generator service that writes reproducible Cellebrite-like export packs into per-dataset folders with `contacts.csv`, `message_threads.csv`, `messages.csv`, `locations.csv`, `locations.json`, `device_locations.plist`, `expected_findings.md`, `manifest.json`, and `README.md`.
- Made every generated dataset clearly fictional and synthetic via manifest/readme notices, synthetic names/identifiers, and expected-findings guidance for offense window timing, central subjects, graph links, location convergence, and QA searches.
- Added deterministic regression tests for seed reproducibility and required output-file generation/manifest coverage.

## Files changed (high-level)
- New standalone utility project: generator app, viewmodel, generator models, and dataset writer service.
- New test project: deterministic generator regressions for reproducibility and required files.
- Repo wiring/docs: solution entry, context whitelist update, this closeout, and ticket completion index entry.
