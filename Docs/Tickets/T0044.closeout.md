# Ticket Closeout: T0044 - Gang documentation packet export v1

## What shipped
- Added `GangDocumentationPacketExportService` with an approved-only preview/read model and HTML packet export path.
- Added a preview-first Gang Documentation UI flow with packet preview, HTML export, and clear export availability messaging for non-approved records.
- Export packets now include subject summary, organization/subgroup context, analyst-entered basis and criteria, supervisor approval metadata, linked supporting evidence references, linked case context, and citations.
- Exported packet content keeps analyst-entered basis, evidence-derived references, and workflow/reviewer metadata in separately labeled sections.
- HTML export writes a `GangDocumentationPacketExported` audit event with documentation id, target id, reviewer metadata, operator, format, and output path.
- Added deterministic regression tests for packet assembly, citation preservation, approval metadata inclusion, approved-only gating, HTML export output, and export audit logging.
