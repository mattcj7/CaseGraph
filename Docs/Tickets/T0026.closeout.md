# Ticket Closeout: T0026 - Locations v1 - LocationObservation ingest + Locations page

## What shipped
- Added `LocationObservationRecord` schema, migration `20260304153000_LocationObservationV1`, and required indexes:
  - `(CaseId, ObservedUtc DESC)`
  - `(CaseId, Latitude, Longitude)`
  - `(CaseId, SubjectType, SubjectId, ObservedUtc DESC)`
- Added queue-backed `LocationsIngest` job type with cancellation-aware progress reporting.
- Implemented locations ingest pipeline:
  - `LocationsIngestJob`
  - `LocationCsvParser` (streaming rows)
  - `LocationJsonParser` (streaming top-level arrays + nested object fallback)
  - `LocationPlistParser` (best-effort dict/array extraction)
- Enforced provenance on every persisted observation:
  - `SourceEvidenceItemId`
  - `SourceLocator` (`csv://...#row=`, `json://...#ptr=`, `plist://...#keyPath=`)
  - `IngestModuleVersion`
- Added `LocationQueryService` with pageable/filterable search and audit logging (`LocationsSearchExecuted`).
- Added `Locations` UI page + `LocationsViewModel`:
  - filters: date range, subject, min/max accuracy, source type
  - paging and DataGrid virtualization
  - row actions: View Source, Copy Citation
- Added queue/audit instrumentation for ingest:
  - `LocationsIngestStarted`
  - `LocationsIngestCompleted`
- Added docs update for data model (`Docs/DATA_MODEL.md`).
